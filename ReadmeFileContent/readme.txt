# 📚 MERN Book Store  

An interactive **online bookstore** built using **MongoDB, Express.js, React, and Node.js (MERN Stack)**. Users can browse books, add them to their cart, and place orders, while admins can manage books and orders.  

## 🚀 Features  

✅ **User Features:**  
- Browse books by category, author, or search 🔍  
- View book details (title, author, price, and description) 📖  
- Add books to cart and place orders 🛒  
- Secure user authentication with JWT 🔐  

✅ **Admin Features:**  
- Add, update, and delete books ✏️  
- Manage user orders 📦  
- Secure admin panel access 🚀  

## 🛠 Tech Stack  

**Frontend:**  
- React.js ⚛️  
- Tailwind CSS 🎨  

**Backend:**  
- Node.js & Express.js 🛠️  
- MongoDB (Mongoose) 🗄️  

**Authentication:**  
- JWT (JSON Web Tokens) 🔐  

**Deployment:**  
- Frontend: Vercel / Netlify 🌎  
- Backend: Render / Heroku 🚀  

## 🔧 Installation & Setup  

1️⃣ **Clone the repository:**  
```bash
git clone https://github.com/your-username/book-store-mern.git
cd book-store-mern
```

2️⃣ **Install dependencies:**  
```bash
# Install frontend & backend dependencies
cd frontend && npm install
cd ../backend && npm install
```

3️⃣ **Set up environment variables:**  
- Create a `.env` file in the `backend` folder  
- Add your **MongoDB URI**, JWT Secret, and other configs  

4️⃣ **Run the project:**  
```bash
# Start backend
cd backend && npm start

# Start frontend
cd frontend && npm start
```
